═══════════════════════════════════════════════════════════════
  🚀 COMANDOS RÁPIDOS PARA SUBIR A GITHUB Y VERCEL
═══════════════════════════════════════════════════════════════

📋 PASO 1: INICIALIZAR GIT
─────────────────────────────────────────────────────────────
git init
git add .
git commit -m "🎨 Initial commit: ImageOptimize converter"


📋 PASO 2: CREAR REPO EN GITHUB
─────────────────────────────────────────────────────────────
1. Ve a: https://github.com/new
2. Repository name: imageoptimize
3. Description: 🎨 Conversor y optimizador de imágenes moderno
4. Public ✅
5. Click "Create repository"


📋 PASO 3: CONECTAR Y SUBIR
─────────────────────────────────────────────────────────────
⚠️ IMPORTANTE: Reemplaza TU-USUARIO con tu nombre de usuario de GitHub

git remote add origin https://github.com/TU-USUARIO/imageoptimize.git
git branch -M main
git push -u origin main


📋 PASO 4: DESPLEGAR EN VERCEL
─────────────────────────────────────────────────────────────
1. Ve a: https://vercel.com/new
2. Inicia sesión con GitHub
3. Click "Import" en tu repo "imageoptimize"
4. Click "Deploy"

¡Listo! ✅
